#WINNIE

> Observe before you judge.  
> Understand your patterns.

Winnie is a fullstack self-awareness dashboard designed to help users track mood, focus, sleep, and productivity, transforming daily inputs into meaningful weekly patterns.

Built with modern technologies and focused first on structure, security, and functionality.

# Features:

- Daily tracking (mood, focus, sleep, productivity)
- Weekly pattern visualization (data-driven insights)
- Dark mode support
- Supabase-backend database integration

# Stacks used:

Frontend
- **Next.js 16** 
- **React 19**
- **TypeScript**
- **Tailwind CSS v4**
- **Framer Motion** 
- **Recharts** 
- **next-themes** 

Backend & Database
- **Supabase**
  - **PostgreSQL database**
  - **Hosted backend services**
  - **Secure API integration**


## Security & Architecture

- Environment variables protected via `.env`
- Supabase client initialized securely
- Type-safe development using TypeScript
- Clean project structure following modern Next.js standards
